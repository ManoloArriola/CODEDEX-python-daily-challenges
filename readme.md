This repository documents my learning journey solving daily coding challenges and converting them into **real-world Python applications**.

Each challenge contains two parts:

- **Challenge Solution** → The original algorithm used to solve the coding challenge.
- **Real Application** → A practical implementation that applies the same logic to solve real problems.

The goal of this lab is to strengthen **algorithmic thinking, Python fundamentals, and practical problem-solving skills**.

---

# Repository Structure

Each project follows a consistent structure:

    challenge_name/
    │
    ├── challenge_solution/
    │   ├── challenge_instructions.md
    │   └── main.py
    │
    ├── real_app/
    │   ├── app.py
    │   └── data.json
    │
    └── README.md


The **challenge_solution** folder focuses on algorithmic logic.  
The **real_app** folder demonstrates how the same logic can solve real-world problems.

# Example Real Applications

Some practical tools developed in this repository include:

- **Exam Evaluation Tool**  
  Automatically grades multiple-choice exams using JSON question banks.

- **Medication Reminder**  
  Calculates medication schedules based on dosage frequency.

- **Dataset Completeness Checker**  
  Identifies missing values in structured datasets.

- **CPU Instruction Decoder**  
  Translates binary instructions into readable operations.

- **KPI Trend Analyzer**  
  Evaluates performance indicators and detects trends.

- **Spam Detection Preprocessor**  
  Cleans and processes text before machine learning analysis.
