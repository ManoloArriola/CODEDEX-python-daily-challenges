# Real World Application

The real application expands the challenge logic into a **Medication Reminder tool**.

The program allows a user to:

- Enter a medication name
- Specify pills per dose
- Select a dosage frequency (6, 8, 12 hours or custom)
- Define the first dose date and time
- Set the treatment duration in days

The program then generates a **complete medication schedule** and calculates:

- Total number of doses
- Total pills required for the treatment
